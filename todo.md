NyayAI - AI-Powered Legal Assistant MVP
Project Overview

NyayAI is a Streamlit-based legal AI assistant for Indian users with legal queries, document summarization, and access to IPC sections.
Core Files to Create (Max 8 files):

    app.py - Main Streamlit application with UI and routing
    database.py - SQLite database setup and IPC sections management
    ai_handler.py - Gemini API integration and AI query processing
    pdf_processor.py - PDF text extraction and summarization
    ui_components.py - Reusable UI components and styling
    config.py - Configuration settings and constants
    requirements.txt - Updated dependencies
    sample_data.py - Sample IPC sections and templates for demo

Features Implementation:

    ✅ Clean Streamlit interface with deep blue theme
    ✅ Legal Q&A with SQLite fallback and AI integration
    ✅ PDF upload and summarization
    ✅ IPC sections database browser
    ✅ Legal draft templates library
    ✅ Chat-like interface for queries
    ✅ Offline fallback capabilities

Design:

    Deep navy blue (#1a1f3a) primary color
    Blue-to-purple gradient accents
    Card-based layout with shadows
    Professional typography
    Responsive design

Tech Stack:

    Streamlit for frontend
    SQLite for data storage
    Gemini API for AI (with offline fallback)
    PyMuPDF for PDF processing
    Custom CSS for styling
