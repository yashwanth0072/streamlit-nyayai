Project Summary

NyayAI is an AI-powered legal assistant designed specifically for Indian users, making legal information more accessible and affordable. The platform provides features such as AI-generated legal query responses, document summarization, and a comprehensive database of Indian Penal Code (IPC) sections and templates. With a user-friendly Streamlit interface, NyayAI enhances legal understanding and aids users in navigating the complexities of legal matters, ultimately striving to simplify the legal process.
Project Module Description

    Legal Q&A: Users can input legal questions and receive AI-generated answers based on IPC sections.
    Document Analysis: Users can upload PDF documents for summarization and analysis.
    IPC Sections Database: A searchable database of key IPC sections with detailed explanations.
    Legal Templates: A collection of ready-to-use legal document templates.
    Interactive Flashcards: Engaging flashcards displaying penal code details and punishment information, with search and filter capabilities.

File Description Inventory

    ai_handler.py: Integrates with the Gemini API for AI responses, manages state, and handles offline fallbacks.
    app.py: The main entry point for the Streamlit application, handling user interface and routing.
    config.py: Contains application configuration settings, including UI styling and database paths.
    database.py: Manages SQLite database operations, including creating tables and populating data.
    nyayai.db: The SQLite database file containing IPC sections, templates, and flashcard data.
    pdf_processor.py: Extracts text from PDF documents and provides analysis capabilities.
    requirements.txt: Lists required Python packages and their versions.
    template_config.json: Configuration file for managing templates (if applicable).
    todo.md: Contains project goals and features to be implemented.
    ui_components.py: Defines reusable UI components and styling for the application.

Technology Stack

    Frontend: Streamlit for creating interactive web applications.
    Backend/AI: Python with Gemini API for AI functionalities.
    Database: SQLite for storing IPC sections, templates, and flashcard data.
    PDF Processing: PyMuPDF for handling PDF files.

Usage

    Install the project dependencies:

    pip install -r requirements.txt

    Run the application:

    streamlit run app.py --server.enableCORS=false
